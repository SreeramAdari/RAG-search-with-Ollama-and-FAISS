# RAG-search-with-Ollama-and-FAISS
This project allows you to **upload a document locally** (`PDF / DOCX / TXT / MD`), generates embeddings using `SentenceTransformers`, indexes them using **FAISS**, and performs **semantic search**.  Then it sends the retrieved context to **Ollama (local LLM)** to generate meaningful answers using a Retrieval-Augmented Generation approach.
